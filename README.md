# IPL-Analysis

1. Project Title
IPL Analysis Dashboard (2008–2025) | Power BI Business Intelligence Project

2. Short Description / Purpose
The IPL Analysis Dashboard (2008–2025) is an interactive Power BI report that provides a comprehensive analysis of the Indian Premier League across multiple seasons. It enables users to explore tournament statistics, team performances, player achievements, and season-wise insights through dynamic visualizations and filters.
The dashboard helps cricket analysts, sports enthusiasts, and decision-makers quickly understand tournament trends, compare team performances, identify top-performing players, and evaluate season outcomes.

3. Tech Stack
The project was developed using the following technologies:
- Power BI Desktop – Dashboard development and interactive reporting
- Power Query – Data cleaning and transformation (ETL)
- DAX (Data Analysis Expressions) – Calculated measures and KPIs
- Microsoft Excel / CSV – Source dataset
- Power BI Visualizations
- KPI Cards
- Tables
- Slicers
- Player Cards
- Custom Images
- Interactive Filters

4. Data Source
The dashboard uses IPL historical match and player statistics from 2008 to 2025.
The dataset includes information such as:
- Match Details
- Season
- Teams
- Champions & Runner-up
- Batting Statistics
- Bowling Statistics
- Player Awards
- Runs
- Wickets
- Boundaries (4s & 6s)
- Venues
- Points Table
Note: This dashboard is created using publicly available IPL statistics for educational and portfolio purposes. It is not affiliated with the IPL governing body or BCCI.

5. Features / Highlights

A. Business Problem
With multiple IPL seasons spanning over a decade, analyzing team performance, player statistics, and tournament outcomes manually becomes difficult.
Sports analysts, broadcasters, fantasy cricket users, and fans need an interactive dashboard that answers questions such as:
- Which team won the tournament each season?
- Who scored the most runs?
- Which bowler took the most wickets?
- Which player hit the most fours and sixes?
- How many matches were played?
- Which teams performed best in the league stage?
- What are the season-wise tournament statistics?
Traditional scorecards and spreadsheets make this analysis time-consuming and less engaging.

B. Goal of the Dashboard
The dashboard is designed to:

- Analyze IPL statistics season-wise.
- Compare team performances.
- Display tournament winners and runner-up teams.
- Highlight Orange Cap and Purple Cap winners.
- Showcase batting and bowling achievements.
- Provide league standings through the points table.
- Offer interactive filtering for different IPL seasons.

D. Business Impact & Insights
The dashboard delivers valuable insights for cricket analysts, broadcasters, franchises, and fans by transforming raw match data into interactive visual reports.

Business Impact
- Provides season-wise tournament summaries in a single dashboard.
- Enables quick comparison of team performance.
- Simplifies player performance analysis.
- Supports sports analytics and media reporting.
- Helps identify batting and bowling trends across seasons.
- Enhances fan engagement through interactive exploration.
- Reduces manual effort required to analyze historical IPL data.
